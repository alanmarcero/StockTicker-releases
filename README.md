# Stonks — Release Feed

Public Sparkle update feed for [Stonks](https://github.com/alanmarcero/StockTicker-macOS) (a macOS menu-bar stock ticker).

- `appcast.xml` is the Sparkle feed the app polls.
- Release binaries are attached as assets on each `vX.Y.Z` release.

The app source lives in a separate private repo; only built releases are published here.
